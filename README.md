# MeuBills

Aplicativo financeiro pessoal/profissional inspirado no Mobills, pensado para uso na web e como PWA instalavel no celular, com suporte offline e sincronizacao posterior.

Status atual: brainstorming e PRD.

## Direcao inicial

- Stack preferida: Vite + React + TypeScript + PWA + Supabase.
- Offline-first no celular, com IndexedDB e fila local de sincronizacao.
- Web/servidor como fonte da verdade em conflitos.
- Sem SEO como requisito relevante para a escolha da stack.
